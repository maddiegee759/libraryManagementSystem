# libraryManagementSystem
Assignment 2 (C++) for CS21003 - Multi-Paradigm Programming

## Task Description
You will develop a C++ program that simulates a Library Management System. The system should allow
users to add and remove books, manage member information, and handle borrowing and returning
operations. Additionally, the program should support file I/O operations to save and load data.

## Guide to Compile
``` shell
# Clone this repo

# Open command prompt

# Navigate to this folder e.g.
cd libraryManagementSystem

# Compile using g++
g++ main.cpp book.h journal.h library.h magazine.h member.h publication.h

# Run .exe file
.\a.exe
```
